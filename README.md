# IS218 Module 3 - Command-Line Calculator

A command-line calculator built with Python using OOP principles, a REPL interface, full test coverage, and GitHub Actions CI.

## Features
- Add, subtract, multiply, divide
- REPL interface — keep calculating until you type `quit`
- Input validation with helpful error messages
- Division by zero handled gracefully
- 100% test coverage enforced in CI

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python3 main.py
```

## Test

```bash
pytest
```
